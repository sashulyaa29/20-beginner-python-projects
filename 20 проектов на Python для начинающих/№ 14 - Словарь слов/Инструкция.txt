Инструкция по расширению словаря:

1. Добавьте новое слово в формате:
   "слово": {
       "Существительное": ["значение1", "значение2"],
       "Глагол": ["значение"]
   }
2. Слова и части речи пишите на русском.
3. Не забывайте ставить запятые между объектами.
4. Файл должен оставаться корректным JSON.
